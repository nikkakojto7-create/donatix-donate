# 💎 DONATIX — Сучасна система автодонату для Garry's Mod

Втомилися від застарілих та нудних донат-меню? **DONATIX** — це нове слово в монетизації вашого ігрового проєкту. Це не просто скрипт, це повноцінна SaaS-екосистема з сучасним інтерфейсом, надійним бекендом та підтримкою криптовалютних платежів.

![Donatix Preview]() ## 🚀 Головні переваги

* **Сучасний UI/UX Дизайн:** Жодних сірих таблиць. Стильне плиткове меню, зручні категорії, бокова панель з профілем та інтегрований інвентар.
* **Крипто-еквайринг з коробки:** Повна інтеграція з **Crypto Pay API** (Telegram). Ваші гравці можуть донатити в USDT, TON, BTC за кілька кліків, а баланс оновлюється миттєво.
* **Абсолютна безпека:** Усі транзакції та перевірки балансу відбуваються виключно на стороні сервера (Backend + NeonDB). Жодної довіри клієнтській частині — нульовий шанс на накрутку.
* **Потужне Lua API:** Створюйте товари як професіонал. Підтримка лімітів, тимчасових послуг (`:SetTerm`), функцій активації (`:OnActivate`) та кастомних перевірок (`:CanBuy`).
* **Власний інвентар:** Гравці можуть купувати предмети зараз, а активувати їх пізніше з особистого сховища.

## 🛠 Можливості API для створення товарів
DONATIX надає розробникам неймовірно гнучкий синтаксис. Створити товар — це справа однієї хвилини:

```lua
DONATIX:AddItem("vip_status")
    :SetName("VIP Статус (30 Днів)")
    :SetDescription("Доступ до VIP професій та унікальної зброї.")
    :SetPrice(50)
    :SetCategory("Привілегії")
    :SetHighlightColor(Color(241, 196, 15)) -- Кастомний колір картки
    :SetIcon("icon16/star.png", false)
    :SetTerm(30)
    :SetCommand("sam setrank %STEAMID% vip")
```
⚙️ Встановлення (Клієнтська частина)
Зареєструйте свій проєкт у веб-панелі donatix.pp.ua.

Отримайте свій унікальний Project ID та Secret Key.

Завантажте цей аддон та помістіть його у папку garrysmod/addons/donatix.

Відкрийте файл lua/donatix/sh_config.lua та вставте ваші ключі.

Готово! Запустіть сервер, і він автоматично з'єднається з веб-панеллю.
